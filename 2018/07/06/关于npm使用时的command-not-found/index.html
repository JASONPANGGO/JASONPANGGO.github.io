关于npm使用时的command not found

Posted on 2018-07-06

曾经好几次，在npm install xxx之后使用xxx提供的命令，被无情提示command not found。

为此，曾试过彻底卸载node.js，走了几次弯路，进了几个坑，仍然没解决问题。

真正的方法是：

此处以webpack为例

  • npm install -g webpack
  • webpack -v之后提示command not found
  • 此时根据自己安装node.js的路径找到node_modules\.bin
  • 一般在C:\Users\ (userName) \AppData\Local内
  • 因为我使用的是yarn，所以我的地址为

C:\Users\Jason\AppData\Local\Yarn\Data\global\node_modules.bin

  • 里面应该可以找到 webpack.cmd
  • 然后复制该地址
  • 打开环境变量，找到系统变量中的Path，然后编辑，把上述地址新增进去。

  • 再次webpack -v
